# Итоговый отчет по проекту поиска и подготовки изображений

Дата: `2026-04-27`

## 1. Что было сделано

В рамках проекта была выполнена полная цепочка подготовки изображений для импорта в каталог:

- отфильтрованы товары без изображений из выгрузок Bitrix;
- по каждому блоку товаров подобраны и проверены внешние источники;
- написаны и адаптированы парсеры под разные сайты и форматы данных;
- выполнена ручная и полуавтоматическая верификация спорных изображений;
- убраны дубли, заглушки, водяные знаки и нерелевантные картинки;
- собраны итоговые папки изображений и подготовлены Excel-файлы для импорта.

Речь шла не о простом “скачивании картинок”, а о многоэтапной работе:

- поиск подходящих источников;
- оценка качества каждого источника;
- настройка сопоставления по артикулам;
- отбраковка мусора;
- пересборка честного остатка после каждой итерации;
- финальная упаковка в импортный формат.

## 6. Сколько реально было источников

В итоговую поставку вошли `22` результативных source-layer без учета `placeholder`.

Это не просто “22 сайта” в лоб:

- часть слоев была официальными сайтами производителей;
- часть — официальными PDF-каталогами;
- часть — розничными магазинами;
- часть — специальными fallback-слоями для точечного добора.

То есть проект включал не только парсинг сайтов, но и работу с несколькими типами источников:

- обычные карточки товаров;
- результаты поиска;
- категорийные листинги;
- PDF-каталоги;
- family-layer для артикульных семейств;
- тонкие точечные источники для единичных позиций.
